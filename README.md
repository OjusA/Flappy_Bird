🐦 Flappy Bird Clone (Python + Pygame)

A simple yet engaging clone of the classic Flappy Bird game built using Python and Pygame. This project demonstrates core game development concepts such as real-time rendering, event handling, collision detection, and physics simulation.

🎮 Features
Smooth bird movement with gravity and flap mechanics
Infinite scrolling pipes with random generation
Real-time score tracking system
Collision detection with pipes and ground
Sound effects for actions (flap, hit, score)
Welcome screen and continuous gameplay loop
🛠️ Tech Stack
Python
Pygame
📂 Project Structure
├── main.py
├── gallery/
│   ├── sprites/
│   ├── Audio/
🚀 How to Run

Install dependencies:

pip install pygame

Run the game:

python main.py

Understanding game loops and frame updates
Implementing physics (gravity, velocity, acceleration)
Handling user input and events
Managing assets like images and sounds
